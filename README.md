## What This Project Does

Demonstrates core R programming concepts using the **gapminder** dataset:

| Topic                 | Script                                            |
| --------------------- | ------------------------------------------------- |
| a. For loop           | `scripts/a_for/`                                  |
| b. While loop         | `scripts/b_while/`                                |
| c. Repeat loop        | `scripts/c_repeat/`                               |
| d. Functions          | `scripts/d_functions/`                            |
| e. Data Visualization | `scripts/e_data_visualization/` + `submission.md` |
| f. Data Science       | `scripts/f_data_science/` + `submission.md`       |

The primary submission document is **`submission.md`**, which knits to a polished
PDF containing all six topics with code, output, and written explanations for
topics e and f.

## How to Run

### 1. Install required packages (run once)

```r
install.packages(c("tidyverse", "here"))
```

### 2. Run individual scripts

```r
source("scripts/a_for/for_1.R")
source("scripts/a_for/for_2.R")
source("scripts/b_while/while_1.R")
source("scripts/b_while/while_2.R")
source("scripts/c_repeat/repeat_1.R")
source("scripts/c_repeat/repeat_2.R")
source("scripts/d_functions/functions_1.R")
source("scripts/d_functions/functions_2.R")
source("scripts/e_data_visualization/data_visualization_1.R")  # saves PNGs to output/
source("scripts/e_data_visualization/data_visualization_2.R")  # saves PNGs to output/
source("scripts/f_data_science/data_science_1.R")
source("scripts/f_data_science/data_science_2.R")
```

## Notes

- No `setwd()` is used anywhere. Scripts use `here::here()` for portable paths.
- All loop bounds and model parameters are defined as named constants.
- Missing values are handled explicitly with `drop_na()` / `na.omit()`.
