# B. While Loop

A `while` loop keeps running as long as a condition is true. Use it when the number of repetitions is not known in advance.

## Examples

### while_1.R — Countdown from 10 to 1
Starts at 10 and subtracts 1 each step until the count hits zero. Demonstrates a `while` loop with a simple decreasing condition.

**Run:**
```r
source("scripts/b_while/while_1.R")
```

### while_2.R — Double a Number Until It Exceeds 1000
Starts at 1 and doubles the value every step. The loop stops itself the moment the value exceeds 1000, without knowing in advance how many steps that takes.

**Run:**
```r
source("scripts/b_while/while_2.R")
```
