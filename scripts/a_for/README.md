# A. For Loop

A `for` loop repeats a block of code a fixed number of times, stepping through each item in a sequence one by one.

## Examples

### for_1.R — Print Numbers and Their Squares
Loops from 1 to 10 and prints each number alongside its square. Demonstrates the basic structure of a `for` loop with a single action per step.

**Run:**
```r
source("scripts/a_for/for_1.R")
```

### for_2.R — Sum All Numbers from 1 to 100
Adds every number from 1 to 100 into a running total using a `for` loop. Verifies the result against the formula `N*(N+1)/2`.

**Run:**
```r
source("scripts/a_for/for_2.R")
```
